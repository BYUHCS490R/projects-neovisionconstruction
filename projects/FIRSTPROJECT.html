Internet

The internet is a big network that connects millions of devices worldwide, letting people share informatiom and communivate with each other

Web

The web is a collection of websites and pages that you open using a web browser, like Chrome or Firefox.

Difference Between Internet and Web

The internet is the system that connects computers, while the web is the information we see online. The internet is like a curry we eat , and the web is like the ingredients that we add.

ASCII and Unicode

ASCII is a simple code that uses numbers to represent basic English letters and symbols.Unicode is a bigger code system that includes letters and symbols from almost every language, plus things like emojis.

UTF-8

UTF-8 is a way to store Unicode characters. It uses less space for simple letters and more space for complex ones.

URL UNIFORM RESOURCE LOCATOR:

Is the web address that you type in the browser

Domain Name

A domain name is the human-friendly name of a website, like google.com, instead of a string of numbers.

HTTP (HyperText Transfer Protocol)

HTTP is the set of rules that lets your browser get information from a website. It is not safe because the data is sent without protection.

IP Address

IP address is a special number that identifies a device on the internet, just like a home address tells where you live.

DNS (Domain Name System)

DNS works like a translator. It changes easy-to-remember website names into the number addresses computers use.

W3C Recommendations

The W3C creates rules so websites work correctly and look good on all browsers and devices.

XML, XHTML, HTML, and HTML5

XML: Used to organize and move data. XHTML: A stricter version of HTML that follows extra rules. HTML: The basic language used to make web pages. HTML5: The latest version of HTML, with support for things like videos and animations.
