Answers:

  1. Header hold the introduction of the page which contains(logs,site tittle, search bar.
  2. Navigation will have the links that can help the viewer to navigate for the information. we can keep navigation in sibe, or footer.
  3. The main element contains the main content unique to that page, such as an article, product details, or dashboard. There should only be one main element per page, and it should not be inside header, footer, or nav.
  4. The footer is at the bottom of the page and usually contains copyright information, legal links, contact details, or secondary navigation links. It can repeat some links found elsewhere on the page.
  5. To make your HTML valid, start with a correct doctype, nest elements properly, use only one main element, provide alt text for images, and add labels for forms. You can check your page with the W3C HTML validator to catch any errors.
