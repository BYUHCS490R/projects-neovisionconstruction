1. What is AJAX?

AJAX lets your webpage talk to a server and get or send data without reloading the page. Example: Google search suggestions appear instantly while you type.

Why put JS in a separate file?

Keeps HTML and JS separate → easier to read and manage. JS can be reused on multiple pages. Browsers can cache the file → faster loading. Makes teamwork easier.
