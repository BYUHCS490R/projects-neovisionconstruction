Answers

1. What kind of photos are good to encode using JPEG?
JPEG is best for normal photos with lots of colors, like pictures of people, nature, or places.

2. What kind of photos are good to encode using GIF?
GIF is good for simple pictures or animations, like logos or cartoons, because it only supports up to 256 colors.

3. What is unique about the PNG format?
PNG keeps the image quality high and supports transparent backgrounds, so it is great for logos or images used on top of other designs.

4. How do you resize your photo to the size it should be displayed?
Open it in a photo editor (like Paint, GIMP, or Photoshop) and change the width and height before saving it.

5. Why should you resize photos using photo editing software instead of CSS?
Because CSS only makes the picture look smaller on the webpage, but the file is still big. Editing software makes the file smaller, which helps the page load faster.
