Questions & Answers

1. What kind of photos are good to encode using JPEG?
JPEG is best for detailed, colorful images like photographs because it supports millions of colors and compresses them efficiently.

2. What kind of photos are good to encode using GIF?
GIF is good for simple images with limited colors, such as logos, icons, or small animations, since it only supports up to 256 colors.

3. What is unique about the PNG format?
PNG supports lossless compression and transparency, making it excellent for high-quality images, graphics, and logos with clear backgrounds.

4. How do you resize your photo to the size it should be displayed?
You can resize photos using image editing software (like Photoshop, GIMP, or Paint) before uploading, or set the desired width and height attributes in the <img> tag.

5. Why should you resize photos using photo editing software instead of resizing it using CSS?
Because resizing in CSS only changes how the image looks on the page, but the full file size is still downloaded. Resizing with photo editing software reduces the actual file size, making the page load faster.
